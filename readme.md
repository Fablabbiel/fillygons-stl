Fillygons
=========

Precompiled fillygon 'stl' files.


Hints for printing
------------------

Try the 0.25mm files first. If the hinges are too tight or loose,
try 0.4mm or 0.25mm files. Which version works best also can depend
on the exact kind of PLA filament used.
